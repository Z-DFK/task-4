    Определение

    GitHub — крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки.

    С помощью GitHub разработчики могут:

    - настраивать свой проект для удаленной работы с другими программистами;
    - публиковать результаты разработки и показывать их другим пользователям;
    - находить и разбирать open-source-проекты, чтобы научиться новому;
    - общаться с другими разработчиками, находить единомышленников.

    На что основан gitHub

    Веб-сервис основан на системе контроля версий Git и разработан на Ruby on Rails и Erlang компанией Github, Inc.

    Аналоги GitHub

    - GitLab — это инструмент для хранения и управления репозиториями Git. Он дает возможность выполнять совместную разработку силами нескольких команд, применять обновления кода и откатывать изменения, если это необходимо. Решение может работать на собственном сервере или в облаке;

    - Bitbucket Cloud — это инструмент для хостинга кода и совместной работы на основе Git, предназначенный для команд. Лучшие в своем классе интеграции Jira и Trello для Bitbucket создают для всей команды разработчиков единое пространство, в котором ее участники вместе работают над проектом;

    - SourceForge — система совместной разработки компьютерных программ. Программное обеспечение SourceForge разрабатывается и продаётся компанией CollabNet. Один из самых больших в мире веб-сайтов для разработчиков открытого программного обеспечения;

    - launch pad — пусковая платформа, стартовая платформа — веб-сайт с системой совместной разработки программного обеспечения, разрабатываемый и обслуживаемый компанией Canonical;

    - Apache Allura — это программное обеспечение с открытым исходным кодом для управления репозиториями исходного кода, отчетами об ошибках, обсуждениями, вики-страницами, блогами и многим другим для любого количества отдельных проектов;

    - Cloud Source — это хостинг проектов, который, по задумке компании, должен найти своё место на одной площадке с другими «облачными» сервисами Google;

    - AWS CodeCommit — это безопасная, хорошо масштабируемая, полностью управляемая служба контроля версий, в которой размещаются частные репозитории Git;

    - DevHub — это программа, которая позволяет вам создавать скретч-организации, разблокированные пакеты и управляемые пакеты второго поколения

    Какие функции выполняет сервис GitHub

    Участникам, кроме непосредственного хранения кода, своих проектов, можно общаться, комментировать изменения друг друга, отслеживать новости знакомых. У программистов есть возможность объединять репозитории и выводить вклад участника в виде дерева.

    Разработка платформы GitHub началась 19.11.2007. Сайт был запущен в апреле 2008 года Томом Престоном-Венером, Крисом Уонстратом, П.Дж. Хайеттом и Скоттом Чаконом после того, как он был доступен в течение нескольких месяцкв в качестве бета-версии
